# IHAgentFrontend — 项目说明

## 项目定位

IHAgentFrontend 是 IHCore 引擎扩展的 **AI Agent 前端层**，负责游戏内的智能决策与操控逻辑。

根据 [Agent架构.md](../Agent架构.md) 的划分，整个系统分为：

- **后端**（本项目不涉及）：共享状态 (GameState)、执行层（原版引擎/脚本）
- **前端**（本项目负责）：战略Agent、调度Agent、介入Agent、玩家指令接口

---

## 范围（Scope）

### 本项目负责

| 模块 | 定位 | 说明 |
|------|------|------|
| **战略Agent** (Strategic Agent) | 军区司令 | 方向性决策，每30秒评估战场态势，输出战略清单 |
| **调度Agent** (Dispatch Agent) | 前线指挥部 | 把战略拆解为具体任务，每5秒调度，响应玩家指令和紧急事件 |
| **介入Agent** (Intervention Agent) | 班长 | 带领指定单位执行具体任务，输出微操指令 |
| **玩家指令接口** (Player Command Interface) | 输入层 | 接收并路由玩家指令（战略级/微操级/全局级） |

### 本项目不负责

- 共享状态 (GameState) 的存储与维护 —— 由后端提供只读/读写接口
- 微操指令的实际执行（IssueOrder/QueueMission）—— 由后端的执行层处理
- 原版引擎的索敌、避障、威胁评估 —— 内置在引擎中，零Token消耗

## 模块职责

### 1. 战略Agent (Strategic Agent)

- **唤醒条件**：每30秒定时 / 玩家战略指令 / 调度Agent上报重大事件
- **输入**：历史操作链（压缩摘要）、战场态势（只读）、上一轮输出、调度反馈、玩家战略指令
- **输出**：`StrategyDirective` — 包含方向(直取/扩张/防守)、优先级权重、全局目标、威胁评估、资源分配建议
- **原则**：只输出"意图"，不输出"指令"

### 2. 调度Agent (Dispatch Agent)

- **唤醒条件**：每5秒定时 / 新战略清单 / 玩家任何指令 / Team状态变化 / 紧急事件
- **输入**：战略清单、战场态势（只读）、上一轮任务列表、介入Agent状态、玩家指令、产线状态
- **输出**：`DispatchPlan` — 包含Team任务列表（单位、脚本、目标、优先级）、产线安排、紧急覆盖
- **原则**：全局指令时可跨Team操作，其余时间只操作Team

### 3. 介入Agent (Intervention Agent)

- **唤醒条件**：调度Agent唤醒（创建/变更/异常）、玩家指令直接指向该Team
- **输入**：操控范围（分配的单位列表）、当前任务、任务流（历时指令链）、本队周边态势
- **输出**：微操指令序列（Move/Attack/Guard等）+ 执行反馈（进度/损失/问题）
- **原则**：仅操作分配的单位，受调度Agent约束

### 4. 玩家指令接口

- 区分指令粒度：战略指令（→异步进入战略Agent）、微操指令（→同步进入调度Agent）
- 玩家指令任何时候都要立即响应，不能等定时周期

---

## 外部接口（对接后端）

本项目通过以下接口与后端（共享状态 + 执行层）交互：

| 接口方向 | 用途 | 频率 |
|---------|------|------|
| **读共享状态** | 查询战场态势、单位状态、产线状态 | 各Agent按自身周期 |
| **写微操指令** | 通过后端执行层下发 IssueOrder/QueueMission | 事件驱动 |
| **读执行反馈** | 获取指令执行结果 | 事件驱动 |

具体接口定义由后端提供，本项目遵循其约定。

---

## 关键原则

1. **职责分离**：战略Agent只出意图，调度Agent拆解任务，介入Agent执行微操
2. **单向依赖**：战略→调度→介入，不反向调用
3. **玩家优先**：玩家指令随时可打断或覆盖Agent决策
4. **权限约束**：介入Agent只能操作分配的单位，不得越权
5. **零副作用的读操作**：战略Agent和态势查询均为只读

## 开发约定

- 使用 C++17/20 标准
- 每个 Agent 对应独立的源文件或模块
- Agent 间的通信优先通过接口/事件回调，避免直接耦合
- 所有 Agent 的输出数据结构化（JSON 或对应的 C++ 结构体）
